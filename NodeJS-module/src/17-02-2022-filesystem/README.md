# Exercício 1

Criar uma pasta com os seguintes arquivos dentro:
 - index.js
 - file.txt
  
Adicionar algum conteúdo no file.txt
Ler o conteúdo do arquivo usando o módulo fs (FileSystem)
 - Os 3 jeitos são:
   - readFile (callback);
   - readFileSync;
   - fs.promises.readFile (.then(callback)).
